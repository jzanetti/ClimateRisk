Welcome to ClimateRisk
#######

**ClimateRisk** is an open-source repository that conducts climate risk 
and adaptation analysis, focusing on the New Zealand transportation sector.

The base model used in **ClimateRisk** is `CLIMADA <https://wcr.ethz.ch/research/climada.html>`_.
Additional New Zealand specifications have been added with a range of options for visualization.

The package is capable of:

- Producing a hazard impact map over areas/properties of interest in New Zealand (e.g., state highway system).

.. note::

   Any issues/suggestions for this system please go to **Sijin ZHANG** at zhans@transport.govt.nz
